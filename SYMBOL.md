# Symbols of Catsplit format


## Symbols
```
<=
Separator of data area and defines area

()
List, Array

(a, b, ..)
Dict, Key-Value Array

, (comma)
Separator of tokens

_ (underscore)
Inherit from parent data structure

* (wildcard)
Allow use Catsplit format in data area
```
